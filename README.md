Random Forest Project

This project uses LendingClub loan data (2007–2010) to predict whether a borrower will fully repay a loan.

Steps
Data exploration and visualization
Handling categorical variables using dummy encoding
Train-test split
Training models:
Decision Tree
Random Forest
Model evaluation using confusion matrix and classification report
Results

Random Forest achieved higher overall accuracy.
Decision Tree performed better in detecting the minority class (not fully paid).

Notes

The dataset is imbalanced, which affects model performance, especially for predicting the minority class.
